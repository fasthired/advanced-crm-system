⚡ QUICK 5-MINUTE SETUP

1. Create Supabase project:
   → Go to supabase.com
   → Click "New Project"
   → Fill in details and wait

2. Get your API keys:
   → Settings → API
   → Copy URL and ANON_KEY

3. Setup database:
   → SQL Editor → New Query
   → Open supabase-setup.sql file
   → Copy entire contents into SQL Editor
   → Click Run

4. Configure app:
   → Create file: .env.local
   → Add your URL and KEY

5. Start running:
   → Terminal: pnpm dev
   → Browser: http://localhost:3000
   → Click: Sign up
   → Done! You're in.
