## 🏃 Quick 5-Minute Getting Started

### If you just want to run it RIGHT NOW:

**Step 1: Create Supabase Project** (1 min)
```
Go to supabase.com → New Project → Save password
```

**Step 2: Get API Keys** (1 min)
```
Settings → API → Copy URL and ANON_KEY
```

**Step 3: Setup Database** (1 min)
```
SQL Editor → New Query → Paste supabase-setup.sql → Run
```

**Step 4: Configure App** (30 sec)
```
Create .env.local:
NEXT_PUBLIC_SUPABASE_URL=your_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_key
```

**Step 5: Run** (30 sec)
```bash
pnpm dev
# Open localhost:3000
# Click "Sign up"
# Done!
```

**That's it! You now have a working CRM in 5 minutes.**
